同源策略是浏览器的一种安全策略，所谓同源是指，域名，协议，端口完全相同。

对！就是这个原理，我们通过服务端请求到数据，然后转发给我们不就好了吗？

CORS

全称是”跨域资源共享”（Cross-origin resource sharing）,

JSONP

利用 <script> 标签没有跨域限制的漏洞，网页可以得到从其他来源动态产生的 JSON 数据。
JSONP 请求一定需要对方的服务器做支持才可以
